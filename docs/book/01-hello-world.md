# Глава 1. Hello, World

Любой разговор о новом языке начинается с программы, которая печатает приветствие. На Mott она выглядит так:

```mott
fnc kort() {
    yazde("Hello, World!")
}
```

Сохрани этот код в файл `hello.mott`, скомпилируй, запусти — и получишь:

```
Hello, World!
```

Дальше — как это всё настроить, и разбор того, что происходит в этих трёх строках.

## Установка компилятора

Компилятор Mott написан на Rust. Чтобы его поставить, у тебя должен быть установлен **Rust** (через [rustup](https://rustup.rs)) и **clang** (обычно уже стоит на macOS/Linux; на Windows — через LLVM-релизы).

```sh
git clone https://github.com/silverhans/mott-lang.git
cd mott-lang
cargo install --path compiler
```

После этого команда `mott` доступна из любой директории:

```sh
$ mott
mott — Mott language compiler (v0.2)

usage: mott <file.mott> [-o OUT] [--emit-c] [--keep-c]
```

Если команда не нашлась — проверь, что `~/.cargo/bin` есть в `$PATH`. `rustup` обычно прописывает это автоматически.

## Первая программа

Создай где-нибудь папку под эксперименты (например, `~/mott-projects/hello/`), положи туда файл `hello.mott` с кодом:

```mott
fnc kort() {
    yazde("Hello, World!")
}
```

Компилируем и запускаем:

```sh
mott hello.mott        # создаёт бинарник ./hello
./hello                # -> Hello, World!
```

Хочешь посмотреть в какой C это транспилируется? Флаг `--emit-c`:

```sh
mott hello.mott --emit-c
```

## Что происходит в этих трёх строках

```mott
fnc kort() {
    yazde("Hello, World!")
}
```

**`fnc`** — ключевое слово объявления функции. В Mott все ключевые слова — из чеченского языка, записанные латиницей. `fnc` — сокращение от функции, читается как "фнц".

**`kort`** — имя функции. `kort` — это чеченское "голова"; в Mott это соглашение для **точки входа** программы. Ровно как `main` в C/Rust/Go. Когда ты запускаешь скомпилированный бинарник, первой вызывается именно `kort`.

**`()`** — список параметров. У `kort` параметров нет, поэтому скобки пустые.

**`{ ... }`** — тело функции.

**`yazde(...)`** — встроенная функция вывода (аналог `print`). Печатает значение и добавляет перевод строки. От чеченского *"язде"* — "напиши".

**`"Hello, World!"`** — строковой литерал, обычное приветствие программиста. Можешь писать внутри кавычек что угодно, на любом языке — строки в Mott это UTF-8: `yazde("Salam, dunya!")` или `yazde("Маршалла хьо!")` работают так же.

## Несколько строк

Можешь добавить больше `yazde`:

```mott
fnc kort() {
    yazde("Hello, World!")
    yazde("Mott — это язык.")
    yazde("Привет из чеченского.")
}
```

Обрати внимание: **точки с запятой не нужны**. Переход на новую строку в Mott — это как `;` в C. Можно ставить `;` если хочется (работает), но канонический стиль — без них.

## VS Code / Cursor

Для удобной подсветки поставь расширение **Mott** из маркета:

1. Открой Extensions panel (`Cmd+Shift+X` / `Ctrl+Shift+X`)
2. Введи в поиск "Mott"
3. Install

После reload окна `.mott` файлы будут красивыми, а в проводнике слева от имени файла появится волк — национальный символ Чечни.

## Попробуй сам

1. Сделай так, чтобы программа печатала три строки: твоё имя, твой любимый цвет, твою любимую цифру.
2. Что произойдёт, если убрать `fnc kort()` и оставить только `yazde(...)`? Запусти, посмотри на ошибку — что она говорит?
3. Что произойдёт, если написать `yazde("Hello")` вне функции `kort`? (Подсказка: в Mott любой код должен быть внутри функции.)
