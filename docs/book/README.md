# Mott — учебник

Пошаговое введение в язык программирования **Mott** (*мотт*, чеч. "язык"). За одну неделю вечерних чтений ты пройдёшь от "что за странный чеченский код" до уверенного написания небольших программ.

## Для кого это

- Ты умеешь программировать хотя бы на одном языке (Python, JS, C — подойдёт любой).
- Тебе интересно как выглядит язык с ключевыми словами на родном для многих, но незнакомом большинству языке.
- Ты хочешь попробовать скомпилировать свой первый `.mott` файл в натив и увидеть как работает компилятор изнутри.

Если ты никогда не программировал — учебник тоже подойдёт, но темп для тебя будет быстрым. Параллельно загляни в любой общий "introduction to programming" курс.

## Формат

Каждая глава — одна тема, один `.md` файл. В конце главы есть мини-задачки ("попробуй сам") и кнопки `← предыдущая | следующая →`.

Код в примерах **весь проверен** — его можно скопировать в файл с расширением `.mott`, скомпилировать командой `mott file.mott` и запустить. Если что-то не работает — это баг учебника, открывай issue.

## Формальная спецификация

Учебник учит "как писать". Если хочешь знать **точные** правила грамматики и семантики — смотри [`../spec.md`](../spec.md). Там EBNF, правила типов, детали лексера.
